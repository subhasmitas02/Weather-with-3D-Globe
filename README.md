# Weather with 3D Globe

A visually immersive web application that displays weather data on a 3D globe using **HTML**, **CSS**, **JavaScript**, **Three.js**, and a **Weather API**. The project provides a dynamic and interactive globe, where users can view current weather conditions for any location on Earth.

## Features

- **Interactive 3D Globe**: A fully interactive globe with rotation and zoom functionality powered by [Three.js].
- **Weather Data**: Real-time weather information including temperature, humidity, wind speed, and weather condition for any location.
- **Weather API**: Integrated with a weather API (e.g., OpenWeatherMap) to fetch real-time weather data.
- **User-Friendly Interface**: Simple design and controls to search for cities and view weather details.
- **Global Location Access**: Navigate to any location on the globe to get weather information for that area.

## Technologies Used

- HTML, CSS, JavaScript, Three.js, Weather API (e.g., OpenWeatherMap).
